# A project to play with Phaser.js

## run

```
npm install
npm run start
```

## build

```
npm run build
```

## memo to init

```
npm init
npm install phaser
npm i --save-dev webpack webpack-cli webpack-dev-server \
                 copy-webpack-plugin html-webpack-plugin style-loader css-loader \
                 typescript ts-loader
```
